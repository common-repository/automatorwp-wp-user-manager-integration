Connect AutomatorWP with WP User Manager

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[WP User Manager](https://wordpress.org/plugins/wp-user-manager/ "WP User Manager") lets you create highly customizable user profiles together with custom user registration, login, password recovery and account customization forms to your WordPress website.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with WP User Manager adding new triggers that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User logs in to the site.
* User registers through any/specific form.
* User changes profile photo.
* User changes profile cover.
* User changes profile description.
* User joins any/specific group.

= Actions =

* Add user to a group.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/wp-user-manager/ "AutomatorWP - WP User Manager") available that let's you expand the integration between AutomatorWP and WP User Manager bringing you more ways to automatize your community.

= What is AutomatorWP? =

AutomatorWP is an automator plugin that lets you connect your WordPress plugins together and add automation to unlimited workflows.

You can automate anything including sales, marketing, administrative tasks, learning and any other kind of processes you want letting you save time and get focused on your most important work.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
